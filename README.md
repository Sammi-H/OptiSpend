# SpendWise 💸

SpendWise is a simple personal finance tracker built as a single-page application (SPA).  
It helps users quickly add expenses and visualize their spending habits in a clear and minimal interface.

---

## 🚀 Features

- Add expenses (amount + category + description)
- Categorize expenses (Food, Home, Transport, Entertainment, Shopping, Other)
- View total monthly spending
- Interactive charts (bar chart + pie chart)
- Transaction history list
- Local storage support (no backend required)

---

## 📊 Data Visualization

- **Bar Chart** → Shows daily spending over time
- **Pie Chart** → Shows spending distribution by category

---

## 🛠 Tech Stack

- Next.js (React framework)
- React Context API (state management)
- Recharts (data visualization)
- LocalStorage (data persistence)
- CSS (styling)

---

## 🎯 Purpose

This project was built to demonstrate:

- Component-based architecture
- State management in React
- Data visualization
- Local data persistence
- Clean and scalable frontend structure
